# FingenNotify

Приложение для пересылки PUSH-уведомлений (например, от банковских приложений) в [Fingen](https://play.google.com/store/apps/details?id=com.yoshione.fingen) — программу для учёта личных финансов. Полезно, когда банк присылает уведомления через PUSH вместо СМС.

Приложение работает полностью локально — не собирает и не отправляет никаких данных в интернет.

## Настройка

### Шаг 1. Разрешить доступ к уведомлениям

Откройте на телефоне: **Настройки → Приложения → Специальный доступ → Доступ к уведомлениям** и включите FingenNotify. Появится предупреждение о том, что приложение сможет читать все уведомления — нажмите «Разрешить».

> На английском: Settings → Apps → Special access → Notification access.

### Шаг 2. Добавить банковские приложения

В окне приложения нажмите меню (три точки) → «Добавить приложение». Заполните два поля:

- **Имя пакета** — техническое название приложения банка (как его узнать — см. ниже).
- **Имя отправителя** — имя, под которым Fingen будет показывать сообщения во «Входящих». Для удобства лучше использовать то же имя, что у SMS-отправителя банка (например, `900` для Сбербанка).

Также можно загрузить готовый список через меню «Импорт пакетов». Пример файла: [example_package/FingenNotify_packages.json](example_package/FingenNotify_packages.json).

## Примеры настроек

| Банк | Имя пакета | Имя отправителя |
|------|-----------|-----------------|
| Сбербанк | ru.sberbankmobile | 900 |
| Альфа-Банк | ru.alfabank.mobile.android | Alfa-Bank |
| МТС Банк | ru.mts.money | MTS-Bank |
| Мир Pay | ru.nspk.mirpay | Mirpay |

## Как узнать имя пакета

### Способ 1. Спросить у знакомых или найти в интернете

Самый простой вариант — найти имя пакета в таблице выше, на форумах (например, 4pda) или спросить у тех, кто уже настраивал приложение.

### Способ 2. Извлечь из APK-файла (для продвинутых)

Если у вас есть APK-файл банковского приложения, имя пакета можно найти внутри него:

- **С помощью Android Studio:** меню Build → Analyze APK → выберите файл. В списке файлов откройте `AndroidManifest.xml` и найдите значение `package=`.
- **С помощью ClassyShark:** скачайте [ClassyShark.jar](https://github.com/google/android-classyshark/releases) (требуется [Java](https://www.java.com/ru/download/)). Откройте APK, найдите `AndroidManifest.xml`, в разделе `<manifest` найдите параметр `package=`.

### Способ 3. Режим захвата всех уведомлений (когда ничего не выходит)

Добавьте приложение с именем пакета **\*** (звёздочка), а имя отправителя — любое. В этом режиме FingenNotify начнёт записывать все уведомления, приходящие на телефон. Дождитесь нужного уведомления от банка, найдите его в списке и скопируйте имя пакета.

> **Внимание:** в этом режиме записываются абсолютно все уведомления (включая системные), поэтому не оставляйте его надолго.

## Безопасность

Исходный код доступен для изучения и предоставляется «как есть».

Не доверяйте готовым APK-файлам из непроверенных источников, особенно если они запрашивают дополнительные или лишние разрешения. Самый надёжный способ — собрать APK самостоятельно из исходного кода с помощью Android Studio.
